Si7021 Temperature & Humidity Sensor Description

The Si7021 is a highly accurate, digital temperature and humidity sensor. The sensor gathers data from the air around it to calculate the temperature and relative humidity. These values are analog at first, but the sensor converts them to digital values before sending them to the ESP8266 Thing. This makes it easier for us to read on the other end.

 

Real World Application

A similar sensor can be found in any home thermostat.

MMA8452 Accelerometer Sensor Description

The MMA8452 is able to tell you its orientation in space with x-, y-, and z-planes that use gravity to tell it its position. For example, when you tilt the sensor left, the bar on the accelerometer bar graph will rise to “Landscape Left”. The accelerometer on display is attached to a motor that will vibrate whenever the orientation is not flat.

 

Real World Application

Accelerometers are more commonly used in smartphones to identify the phone’s orientation in space for GPS, compass apps, and rotates the phone’s screen when the physical phone is turned.

TEMT6000 Ambient Light Sensor Description

The ambient light sensor uses a photosensitive resistor to detect the intensity of light in a room. This sensor is attached to a separate smart light bulb that will dim or brighten with the correlating amount of light detected.

Real World Application

The sensor is commonly used in mobile phones, computers and other devices to control backlight dimming.

APDS-9600 RGB/Gesture Sensor Description

The RGB/Gesture sensor detects the amount of ambient light, measures color values, and reads gestures. If you swipe your hand left over the sensor, the bar graph on the monitor will display “left”. The gesture sensor on display is attached to a smart light bulb that will either turn on or off for varying gestures.

 

Real World Application

The same exact sensor is used in Samsung Galaxy phones to detect gestures from the user.

MB1043 HRLV-MaxSonar-EZ4 Description

The MB1043 HRLV-MaxSonar-EZ4 sensor, or the “Tophat” sensor, uses ultrasonic sound waves to detect how far the object in front of it is. It does this using an internal clock the starts when the sound wave is sent out and ends when the wave returns to the sensor. This is similar to how a bat uses echolocation to detect where its prey is.

 

Real World Application

TThis sensor can be used as a radar gun to measure the speed of an object that is traveling parallel to the sensor (by measuring distance over time). .
